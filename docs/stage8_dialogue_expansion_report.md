## 文风原则

- 只补短句，不补长篇设定说明。
- 用角色反应承接画面，不让 CG 变成连续幻灯片。
- 龙灵继续保持残响式提示，不直接解释机制。
- 赵无炎救赎线避免洗白，强调活着偿还。
- 坏结局保留冷感，不做灾变百科式描述。
