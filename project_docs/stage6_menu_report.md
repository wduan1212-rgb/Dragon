# Stage 6 Menu Report

## 主菜单调整

- 主菜单改为更干净的进入界面。
- 按钮收束为：开始游戏、图鉴、设置、退出。
- 移除读取进度按钮，降低当前版本入口复杂度。
- 主菜单背景已改为优先使用 `images/bg/title_bg_new.png` 静态新图，缺失时回退旧静态背景。
- 主菜单不调用剧情过场，不显示 HUD、章节卡或对话框。

## 入口行为

| 按钮 | 行为 |
|---|---|
| 开始游戏 | `Start()`，进入角色选择 |
| 图鉴 | `ShowMenu("gallery")` |
| 设置 | `ShowMenu("preferences")` |
| 退出 | `Quit(confirm=True)` |

## 稳定性

所有主菜单音频都走 `safe_bgm` / `safe_sfx`，缺失时不会阻塞启动。第六阶段追加修复后，主菜单不再使用首页视频背景。
